# 文档中心

本页是面向赛事评委与赛题企业代表的评审资料索引。本项目由三名学生组成的参赛团队独立完成，目标是帮助企业解决新品研发决策问题；材料按业务价值、技术证据、试点验收和治理边界组织，而不是按团队开发过程组织。

| 评审材料 | 面向角色 | 主要回答的问题 |
|---|---|---|
| [评委快速指南](00_评委快速指南.md) | 综合评审 | 3 分钟理解价值、差异、证据和边界 |
| [行业与业务分析](01_行业分析报告/README.md) | 企业业务评审 | 问题依据是否可核验、哪些假设仍待企业验证 |
| [技术白皮书](02_技术白皮书/AI产品开发引擎技术白皮书.md) | 业务与技术评审 | 架构、算法、数据、验收和落地设计是否完整 |
| [回测实验报告](03_回测实验报告/README.md) | 技术评审 | 协议、结果和防泄漏证据能否复现 |
| [进化与 DFM](04_进化式创意引擎/README.md) | 产品与供应链评审 | 多目标搜索与制造规则的边界是什么 |
| [现场评审演示导览](05_Demo演示/README.md) | 综合评审 | 6 分钟演示将呈现哪些证据 |
| [参考文献](06_参考文献/README.md) | 全体评审 | 业务、方法和治理依据来自哪里 |
| [安全治理](07_安全治理与生产化.md) | 安全与治理评审 | 威胁、控制与上线门禁是否充分 |
| [架构决策](08_架构决策记录.md) | 企业技术评审 | 关键技术取舍、理由与后果是什么 |
