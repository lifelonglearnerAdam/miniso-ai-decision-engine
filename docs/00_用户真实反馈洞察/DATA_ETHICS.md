# 数据伦理与使用边界

## 采集边界

- 仅采集公开可见的标题、链接、互动量、正文摘要和评论主题；
- 小红书由账号持有人本人在本机 Codex 内置浏览器完成登录；
- 未读取或保存密码、验证码、Cookie、私信、通讯录和非公开个人信息；
- 未执行点赞、评论、关注、私信、发布或账号设置修改；
- 不批量转载完整评论文本，不保存评论者身份画像。

## 证据等级

- A：详情页正文和/或公开评论可见；
- B：正文摘要、评价聚合或搜索卡片信息较完整；
- C：仅搜索摘要或主题页片段，只适合发现线索。

## 分析限制

1. 样本为目的性采样，不代表平台全量用户或消费者总体；
2. 搜索和推荐结果受平台算法、登录状态、地区和时间影响；
3. Trustpilot、BBB 等投诉型评价页存在负向自选择；
4. 不同平台的互动量定义不同，不能直接横向相加；
5. 情绪、主题和严重度为研究编码，适合辅助决策，不应替代人工复核；
6. 公开链接可能删除、变更或要求登录，建议在重要决策前再次核验。

## 合理使用建议

- 对外展示以主题归纳和聚合统计为主；
- 引用单条样本时保留原始链接和证据等级；
- 不以单条负面评价推断品牌整体表现；
- 不用于识别、联系、骚扰或画像具体用户；
- 商业部署前应由法务、隐私和平台合规团队复核采集方案。
