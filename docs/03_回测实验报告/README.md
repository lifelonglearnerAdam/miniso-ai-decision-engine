# 时光机回测实验报告

## 结论先行

当前实验是**可复现的合成数据验证**，用于证明固定训练窗、隔离窗、滚动步长、结果列隔离、三基线和排序指标能够正确运行。它不是名创优品真实历史回测，不可作为生产 KPI 或 ROI 证据。

## 协议

| 参数 | 值 |
|---|---:|
| 合成产品决策记录 | 2,400 |
| 固定训练窗 | 180 天 |
| 隔离窗 | 14 天 |
| 预测窗 | 90 天 |
| 滚动步长 | 30 天 |
| 完成窗口 | 15 |
| 随机种子 | 2026 |

候选评分前，引擎删除 `is_hit`、`sales_90d` 和 `realized_margin`；每个窗口的日期、样本量、正例数和受保护列写入 [metrics.json](../../artifacts/demo/metrics.json)。

## 固定种子结果

| 方法 | Precision@20 | Recall@20 | Lift@20 | NDCG@20 |
|---|---:|---:|---:|---:|
| 稳定随机排序 | 0.250 | 0.058 | 0.875 | 0.232 |
| 历史品类率 | 0.317 | 0.075 | 1.133 | 0.320 |
| 可解释表格模型 | **0.490** | **0.117** | **1.752** | **0.471** |

## 企业证据形成路径

1. 由业务确认候选宇宙、爆品标签、K、观察窗与退货/缺货处理；
2. 冻结字段可用时间、SKU 族、基线和最终盲测期；
3. 只使用截止日已到达的数据，统计预处理在训练窗内拟合；
4. 报告逐窗与分组结果、区间、失败窗、错误案例和运行成本；
5. 超过业务基线后进入影子运行，而非直接自动执行。
