# AI 产品开发引擎技术白皮书

本目录保存三人学生参赛团队向赛事评委与赛题企业提交的可版本化解决方案白皮书。正式评审交付件同时提供 Markdown、Word 与 PDF；GitHub Release 中也提供统一版本化下载附件：

- [Markdown 源稿](AI产品开发引擎技术白皮书.md)
- [Word 评审版](../../output/docx/AI产品开发引擎技术白皮书.docx)
- [PDF 评审版](../../output/pdf/AI产品开发引擎技术白皮书.pdf)

白皮书采用“实现、合成演示、企业待验证”三级证据口径；任何合成结果均不得解释为名创优品真实业务成绩。
