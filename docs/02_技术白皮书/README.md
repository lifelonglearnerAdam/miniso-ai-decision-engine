## 架构亮点

### 1. 多 Agent 协作
```
Trend Agent → Creative Agent → Review Agent → DFM Agent → Panel
     ↑              ↑               ↑            ↑          ↑
  (社媒数据)   (趋势→创意)      (多维度评审)  (可制造性)  (消费者验证)
```

### 2. 时光机回测 Protocol

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 训练窗口 | 180天 | 历史数据训练期 |
| 空白间隔 | 14天 | 模拟"未来不可知" |
| 预测窗口 | 90天 | 预测范围 |
| 滚动步长 | 30天 | 时间窗移动步长 |

### 3. 虚拟面板校准三阶段

1. **锚定回归**: 用少量真实评分校准 LLM 输出
2. **方差修正**: Bootstrap 估计置信区间
3. **保形预测**: 无分布假设的预测区间
