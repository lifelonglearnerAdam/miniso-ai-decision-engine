### 一句话结论
 名创优品正在从“高性价比杂货店”升级为“以 IP 设计和全球门店网络驱动的价值零售平台”。2025 年和 2026 年一季度
 的增长显著快于中国整体零售环境，但增长质量需要结合海外直营投入、IP 授权成本、调整后利润与自由现金流共同

1. 行业口径：名创优品并非单一日用品零售商，而是价值零售、生活方式消费、IP 授权商品和潮玩四类赛道的交叉公

2. 需求结构：2025 年中国社零增长 3.7%，但体育娱乐用品和文化办公用品分别增长 15.7%和 17.3%，IP 授权消费也

3. 增长来源：2025 年收入增长由中国 MINISO、海外 MINISO 和 TOP TOY 共同推动；海外收入占 MINISO 品牌收入

提升了收入弹性，也提高租金、人工和资本开支。

5. 盈利判断：2025 年毛利率稳定、自由现金流增长，但报告净利润受永辉相关损失及金融工具影响明显。分析公司
经营质量时，应优先看调整后经营利润、调整后净利润与现金流。

6. 核心风险：IP 热度波动、海外经营与关税/汇率、直营店回报、加盟与代理治理、产品质量和库存、非核心投资造

## 03 市场环境：零售与 IP 消费

### 3.1 中国零售：总量温和、品类分化

2025 年中国社会消费品零售总额为 50.12 万亿元，同比增长 3.7%；商品零售增长 3.8%，网上零售增长 8.6%。限额
以上业态中，便利店和超市分别增长 5.5%与 4.3%，专业店增长 2.6%，品牌专卖店下降 0.6%。这表明线下零售并非
普遍繁荣，能够持续增长的企业需要依靠品类、产品力、渠道效率或门店体验获得结构性份额。

### 分析含义
 IP 并非“免费流量”。它提高产品辨识度、客单与复购的同时，也带来授权费、开发周期、合规审核和热度衰减风险。名
 创优品 2025 年授权费用增长 44.6%，快于收入增长 26.2%，说明 IP 投入正在加深，后续应跟踪其对毛利、周转和爆
 款持续性的贡献。

          案例判断

        球门店 5,891 家，其中日本 4,784 家、海外 1,107 家；销售额连续 15 年创新高。较上一年度披露的 7,242
        亿日元和 5,670 家门店，销售额约增长 6.5%，门店数约增长 3.9%。[C1][C2]

### D.2.3 可复制点与不可复制点
        • 可复制：用清晰的“核心价格带—升级价格带—IP 价格带”管理商品，而不是把所有商品放在同一价值叙事中。

          MINISO 应坚持“设计感可负担”的核心价格承诺，同时把高溢价收藏、限量与盲盒更多留给 TOP TOY。对于大
          店，建议采用“基础刚需区—趋势新品区—IP 主题区—高客单体验区”的分层，而非全店 IP 化。

### D.5.3 增长质量与风险

    • IP 集中风险：THE MONSTERS 约占收入 38.1%。超级 IP 可以显著提高增长，但热度变化会放大收入、库存
      和门店客流波动。
    • 品类集中风险：毛绒占收入 50.4%，意味着公司虽然摆脱单一手办形态，却形成新的品类集中。
    • 库存风险：存货由 2024 年末 15.25 亿元升至 2025 年末 54.73 亿元，存货周转天数由 102 天升至 123 天，

      用品零售。
    • 全球化风险：地标旗舰店能够建立影响力，但租金、物流、人才和本地监管会提高固定成本；爆款驱动的需
      求预测难度也更高。

### D.7 对名创优品的战略建议

         1. 明确 MINISO 与 TOP TOY 的品牌边界
         MINISO 以大众生活方式和“可负担的设计/IP 日用品”为中心；TOP TOY 以专业潮玩、收藏和自有 IP 为中心。两
         者可共享供应链、选址和后台，但前台商品、定价、会员内容和店内叙事应清晰区分。

         2. 为 IP 组合建立集中度上限与退出机制
         按单一 IP 收入占比、单一品类占比、授权期限、毛利、库存和社媒热度管理。爆款追加应同时设置最晚下单日
         和清货预案，避免热度下降后形成长尾库存。

         3. 把非 IP 英雄产品作为长期品牌资产
         每个核心品类选择少量可持续迭代的产品系列，连续追踪复购、评分、质量、退货和跨市场表现。IP 联名可以
         为英雄产品引流，但不应替代产品功能和质量。

         4. 海外直营扩张采用门店批次治理
         按国家、城市和开店年份建立门店队列，追踪开业投资、首年销售、成熟期贡献利润、租售比、现金回收期和
         关店率。未达到门槛的市场应优先优化或使用伙伴模式，而不是继续追求数量。

         5. 由“全球供货”升级为“全球商品平台”
         在核心海外市场建立本地商品经理和快速测试机制，将本地需求、价格和法规反馈到产品开发；同时利用中国
         供应链做规模化生产，实现“本地洞察—全球能力”的组合。

         6. 让大店承担品牌与数据任务
         旗舰店和大店不应只追求面积与打卡，应承担新品测试、会员获取、IP 活动和高客单品类验证。对其评价应同
         时看品牌曝光与可量化的销售/现金回报。

         7. 建立竞争情报的固定节奏
         每季度更新 Daiso 价格和门店组合、MUJI 中国同店与本地商品、Five Below 同店与新店回收、泡泡玛特 IP 集
         中与库存周转，并与名创优品内部指标形成同一仪表板。

### D.9 结论

四个案例共同说明，生活方式与潮流零售的可持续增长必须同时具备“商品新鲜感”和“运营纪律”。Daiso 的
规模、MUJI 的一致性、Five Below 的单店回报、泡泡玛特的 IP 能力分别解决了不同问题。名创优品的独
特优势在于能够把中国供应链、广谱生活用品、全球伙伴网络和 IP 联名结合起来；其下一阶段的关键不
是继续扩大可比公司范围，而是把这四种能力转化为清晰的品牌边界、可量化的门店回报和可分散的 IP
组合。

因此，评估名创优品竞争力时，应优先关注：非 IP 基础品是否产生复购，海外直营店是否按批次实现现
金回收，TOP TOY 是否形成可持续自有 IP，以及授权 IP 是否在扣除授权费、营销和库存后仍创造增量利
润。上述四项比单纯门店数和上新数量更能判断增长质量。
