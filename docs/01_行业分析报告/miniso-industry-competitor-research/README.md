## 核心结论

名创优品正在从“高性价比杂货店”升级为“以 IP 设计和全球门店网络驱动的价值零售平台”。报告认为，评估其增长质量时，需要同时观察海外直营投入、IP 授权成本、调整后利润、自由现金流和单店回报。

## 竞品研究

报告选取四类能力标杆：

- **Daiso**：低价规模、SKU 广度与门店标准化
- **MUJI**：生活方式品牌、统一审美与长期品牌资产
- **Five Below**：年轻化价值零售与可复制单店模型
- **泡泡玛特**：自有 IP、内容运营与收藏生态
