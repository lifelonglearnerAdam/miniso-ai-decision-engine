# 行业与业务问题分析

## 可核验业务背景

MINISO 2025 年年报披露，MINISO 品牌在截至 2025 年 12 月 31 日的财年平均每月推出约 1,600 个 SKU；截至期末拥有 300 余人的内部设计团队及多个外部设计团队。年报同时把持续创新、成功上新、及时识别消费者偏好以及新产品市场不确定性列为关键经营能力和风险。[官方来源](https://ir.miniso.com/image/Annual+Report+2025+US.pdf)

这意味着产品研发 AI 的价值点不是“替代创意团队”，而是帮助高频决策形成更一致的证据、约束和复盘闭环。

## 待企业验证的业务假设

| 假设 | 需要的数据 | 反证条件 |
|---|---|---|
| 某些社媒/搜索信号稳定领先销量 | 带事件与入库时间的信号、SKU/品类销售 | 领先关系只在单期成立或反向显著 |
| 结构化概念评分能优于现有业务基线 | 历史候选全集、人工选择、后续结果 | 盲测不优于规则/人工历史选择 |
| DFM 前置能减少低价值打样 | 打样、驳回原因、成本、供应商能力 | 规则误杀高价值概念或无节省 |
| 校准面板能提供增量弱信号 | 真实调研/行为与后续销量 | ECE、覆盖率或分组偏差不达标 |
| 偏好记录能改善后续排序 | 采纳/驳回/修改及原因 | 反馈受曝光选择偏差主导 |

## 竞品分析原则

项目不在缺少同口径证据时制作“价格、SKU、周期、命中率”横向排名。正式竞品研究应：

1. 预先定义品牌、区域、期间与商品口径；
2. 优先使用年报、招股书、官方发布和可追溯数据；
3. 区分自营/加盟、MINISO/TOP TOY、SKU 库存与上新数量；
4. 对抓取数据记录许可、时间戳、缺失和覆盖；
5. 将商业假设与已证实事实分栏呈现。
