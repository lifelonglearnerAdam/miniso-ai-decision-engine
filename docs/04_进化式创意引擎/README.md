# 进化式创意与 DFM 引擎

## 核心设计

引擎同时优化三个目标：需求潜力、可制造性和相对种群的新颖性。若概念 A 在所有目标上不差于 B 且至少一个目标更优，则 A 支配 B；最终返回真正的非支配前沿。

```text
初始模板 → 扩展为完整种群 → 三目标评估
       → 锦标赛选择 → 交叉 → 变异 → 精英保留
       → 下一代 → 非支配筛选 → 产品经理选择
```

## 已修正的原型问题

- 单模板也会扩展到配置种群大小；
- 每代保持完整人口，不再交叉后只剩一半；
- 新颖性使用 Jaccard 距离，不再恒为零；
- 特征从双亲合并，而不是只复制一个亲本；
- 结果通过支配关系筛选，不把综合分冒充帕累托；
- 所有随机过程使用显式种子。

## DFM 规则

当前规则是演示级检查项：稀有材料、低价 OLED、数码防水工艺、高价概念价值点不足、食品接触材料证明、低价 IP 授权费空间。规则返回 ID 和原因，作用是创建待核验任务，不替代工程、质量或法规专家。

企业版本每条规则应包含 owner、品类/地区范围、版本、生效日期、证据、例外审批和命中统计。

## 偏好飞轮的证据边界

当前代码实现三目标权重更新，尚未实现 DPO 或奖励模型训练。采纳/驳回数据投入训练前，需要处理选择偏差、标注一致性、少数意见、隐私与回归评估。
