## 演示流程（5-8分钟）

### 场景：时光机回测

**核心概念**: 穿越回 2024 年 1 月 1 日，系统用当时能获取的数据预测未来 90 天内的爆品，然后验证预测是否准确。

### 演示脚本

#### Step 1: 启动系统 (30s)
```bash
python src/pipeline/run_all.py
```

#### Step 2: 展示回测结果 (2min)
- 展示 Precision@20 曲线
- 展示 Lift vs 随机基线
- 展示 NDCG 柱状图

#### Step 3: 展示校准效果 (1min)
- 展示锚定回归效果
- 展示置信区间

#### Step 4: 展示进化引擎 (1.5min)
- 展示帕累托前沿
- 展示偏好飞轮调整

#### Step 5: 展示 Granger 因果 (1min)
- 证明社媒趋势能预测销量
